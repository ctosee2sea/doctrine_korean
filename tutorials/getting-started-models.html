시작하기 : 모델 우선¶

Note

개발 워크 플로우

여러분이 코드 우선 개발을 할 때는, 객체 개발을 먼저 하고나서, 데이터베이스에 매핑합니다.
여러분이 모델 우선 개발을 할 때에는 UML 등의 툴을 사용해서 애플리케이션을 모델링하고, 이걸 기반으로 데이터베이스 스키마와 PHP 코드를 작성합니다.
여러분이 데이터베이스 우선 개발을 할 때는, 이미 데이터베이스 스키마가 준비된 상태에서 상응하는 PHP code를 만들겁니다.

Note

이 시작 가이드는 개발 중입니다.

모델을 높은 수준으로 설명하기 위해 사용하는 UML 같은 어플리케이션들이 있습니다.
모델링 툴은 구조 방식을 설명하는 엑셀이나 XMl 혹은 CSV 파일로도 만들 수 있습니다.
여러분의 어플리케이션이 모델링 툴을 사용하고 있다면, 개발 워크 플로우로 독트린의 모델 우선 접근이라 불리는 방법을 사용할 수 있습니다.
이 워크플로우에서는 여러분이 모델 설명을 변경하면, PHP 코드와 데이터베이스 스키마를 재생성합니다.
